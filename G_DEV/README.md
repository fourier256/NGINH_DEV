# 로드파이터 리부트

React + TypeScript + Vite 기반의 세로 모드 웹 레이싱 게임 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

## 구현 범위

- 메인 메뉴: 출발, 튜닝, 쇼핑, 헌팅
- 월드맵: 트랙/바다/다리/설산/야간트랙 5개 스테이지
- 로그라이크 카드: 10종 중 3장 랜덤 제시 및 선택
- 레이싱 캔버스: 20km 절차적 도로, 의사 3D 투영, 장애물/아이템/AI 차량 렌더링
- 물리: 자동 가속, 좌우 조작, 부스터, 연료, 슬립/스핀/충돌 처리
- UI: 미니맵, 연료/속도계, 동승자 리액션, 부스터 집중선/화염

## 에셋 정책

요청에는 인터넷 이미지 다운로드가 포함되어 있었지만, 저작권과 라이선스가 불명확한 핀터레스트/상용 게임 에셋은 포함하지 않았습니다. 대신 CSS와 Canvas로 직접 제작한 오리지널 프로토타입 그래픽을 사용해 바로 실행 가능하게 구성했습니다.
